## Design Context

### Users
한국 중·고등학생 (13~19세). 교실에서 또는 이동 중에 스마트폰으로 빠르게 당일 시간표와 학사일정을 확인하는 상황. 정보를 빠르게 훑고 닫는 패턴으로 사용.

### Brand Personality
**신뢰감 있고, 깔끔하며, 방해하지 않는.**
- 필요한 정보를 즉시 보여주고 물러나는 도구
- 군더더기 없이 핵심만
- 학생들이 매일 쓰는 앱답게 피로감 없는 디자인

### Aesthetic Direction
- iOS 시스템 앱 수준의 정제된 미니멀리즘
- 배경: 라이트 모드 `#f2f2f7` (iOS gray), 다크 모드 `#1c1c1e`
- 흰색(라이트) / `#2c2c2e`(다크) 카드, 미세한 그림자
- 주 액센트: `blue-500` (`#3b82f6`) — 현재 수업 강조, CTA 버튼
- 과목별 컬러 서클은 유지 (시각적 구분 역할)
- 폰트: 시스템 폰트 스택 (`-apple-system`, `BlinkMacSystemFont`, `sans-serif`) — 별도 웹폰트 불필요
- 현재 UI 수준 유지 + 다크 모드 추가가 주 과제

**Anti-references:** 화려한 그라데이션, 과도한 애니메이션, 카드 안의 카드 중첩, 알림 배지 남발

### Design Principles
1. **정보 우선** — 장식보다 정보가 먼저. 학생이 3초 안에 오늘 수업을 파악할 수 있어야 한다.
2. **다크 모드 동등성** — 라이트/다크 모드 모두 동일한 정보 밀도와 가독성을 유지한다. `dark:` 클래스로 시스템 설정 자동 연동.
3. **미니멀한 상태 표현** — 현재 수업은 ring 강조, 지난 수업은 opacity 감소. 새로운 시각 요소보다 기존 패턴의 일관성을 유지.
4. **터치 친화성** — 최소 터치 타겟 44px. 모달 배경 탭으로 닫기 등 기대되는 모바일 패턴 준수.
5. **불필요한 인터랙션 없음** — 학생이 매일 반복 사용하는 앱이므로 온보딩·애니메이션 등은 절제.
